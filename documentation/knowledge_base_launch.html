Launching the Knowledge Base

The Knowledge Base can be launched by adding the following parameters and nodes to your launch file:

{% highlight xml linenos=table %} {% endhighlight %}

Nodes

rosplan_knowledge_base
The Knowledge Base Node stores the PDDL model. It can be queried for information on the PDDL domain model, and problem instance. It should be updated by state estimation.

rosplan_scene_database and rosplan_scene_message_store
These nodes are ROS interfaces to a MongoDB, maintained by the University of Birmingham. mongodb_store on ROS wiki.

Parameters

data_path
Path to place automatically generated files, such as plan files.

domain_path
Path to the PDDL domain file.
