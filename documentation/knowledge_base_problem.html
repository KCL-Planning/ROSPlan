Current instances

Topic: /kcl_rosplan/get_current_instances
Message: rosplan_knowledge_msgs/GetInstanceService

{% highlight python linenos=table %} string type_name --- string[] instances {% endhighlight %}

Returns a list of object instance names of the given type. If the type is left empty, then all object instances are returned.

Current knowledge

Topic: /kcl_rosplan/get_current_knowledge
Message: rosplan_knowledge_msgs/GetAttributeService

{% highlight python linenos=table %}string predicate_name --- rosplan_knowledge_msgs/KnowledgeItem[] attributes {% endhighlight %}
